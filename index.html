Frequently Asked Questions

Everything you need to know about PolePosition

How do I set up a season?

Use the /create-season command to start the setup walkthrough. You'll select the number of rounds, choose circuits for each race, and configure your manager and steward roles. The entire process takes about 5 minutes.

What are Manager and Steward roles?

Managers can enter race results, manage teams, and post attendance sheets. Stewards can assign penalties and resolve incident reports. Both roles must be configured for full functionality.

How do penalties work?

Stewards use the /assign-penalty command to assign time or point penalties to drivers. When penalties are applied, driver and constructor standings automatically recalculate based on the new lap times or point deductions.

Can drivers report incidents?

Yes! Drivers use /report-incident to submit incident reports with context and evidence links. Stewards can view all reports, resolve them with penalties, or close them without action.

What points system is used?

The bot uses the 2026 F1 points system: 1st=25pts, 2nd=18, 3rd=15, 4th=12, 5th=10, 6th=8, 7th=6, 8th=4, 9th=2, 10th=1.

Is my data safe?

All season data is stored in your own database. The bot never shares your league information. You have complete control over all data and can delete seasons anytime.

Can I use this for multiple seasons?

Absolutely! Create a new season whenever you want. Each season is completely independent with its own data and standings.

What permissions does the bot need?

The bot needs basic permissions to read/send messages and create channels. Only guild administrators can run setup commands.
